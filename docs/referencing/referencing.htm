Referencing Variables

If an SVG attribute accepts a value type that CSS variables can represent, such as lenght, color, number or percentage, we can use the CSS var() and calc() functions as an attribute value:

<svg id="svgRoot" width="300" height="300" viewBox="-150 -150 300 300" 
	style="border:1px solid silver ; --posY:30px" >
<style>
	:root { --posX : -30px }
	</style>
<circle cx="var(--posX)" cy="var(--posY)" r="120" fill="none" stroke="black" />
</svg>
