A 3D funktion viewer based on HTML and SVG

3D Viewer

SVG only supports a 2D space, therefore a 3D scene must be projected onto a 2D plane. To render 3D scene on a 2D plane, we need to transform every (x,y,z) coordinate into a (x′,y′) SVG coordinate. To rotate the scene, we multiply our coordinates by rotation matrices. If we want to rotate around the Y-axis (left/right) and X-axis (up/down), we use these transformations:

Ry(θ)= [ cosθ 0 sinθ 0 1 0 −sinθ 0 cosθ ] Rx(φ)= [ 1 0 0 0 cosφ −sinφ 0 sinφ cosφ ]

A simple orthographic view ignores the resulting z-coordinates, things don't get smaller in the distance. For a perspective view, we simply divide x and y by z to simulate depth.

Implementation Notes

State

The state variable carries the view state members:

let state = {
	revX : 0.5 ,	// Rotation around X
	revY : 0.5 ,	// Rotation around Y
	zoom : 1 ,
	isDragging : false ,
	lastMouse : { x : 0 , y : 0 }
	} ;

Project()

function project( x, y, z ) {
	// Zoom
	x *= state.zoom; y *= state.zoom; z *= state.zoom;

	// Rotate around X
	let x1 = x * Math.cos(state.revY) + z * Math.sin(state.revY);
	let z1 = -x * Math.sin(state.revY) + z * Math.cos(state.revY);

	// Rotate around Y
	let y2 = y * Math.cos(state.revX) - z1 * Math.sin(state.revX);
	let z2 = y * Math.sin(state.revX) + z1 * Math.cos(state.revX);

	// Return projection point
	return { x : x1 , y : -y2 };   // SVG Y-axis is inverted
	}
