Intersection of Line Segments

A line segment L1 between the points P11 and P12 is define by two vectors:

L1 = p⇀12 − p⇀11

with p⇀=OP, O=(0,0) the origin of the coordinate system, and P=(xp,yp) a point with its coordinates. Any point on this line segment can be expressed as a function of the factor k∈ℝ:

P1 = p⇀11 + k1 ( p⇀12 − p⇀11 )

with 0≤k1≤1. If two lines L1, L2 intersect, they have a point in common:

P1 = P2 p⇀11 + k1 ( p⇀12 − p⇀11 ) = p⇀21 + k2 ( p⇀22 − p⇀21 )

The translates into the equivalent system of two linear equations in two variables k1 and k2:

x11 + k1 ( x12 − x11 ) = x21 + k2 ( x22 − x21 ) y11 + k1 ( y12 − y11 ) = y21 + k2 ( y22 − y21 )

Now we separate variables from constants:

k1 ( x12 − x11 ) − k2 ( x22 − x21 ) = x21 − x11 k1 ( y12 − y11 ) − k2 ( y22 − y21 ) = y21 − y11

To make that more manageable, we subsitute

A=x12−x11 B=x22−x21 C=y12−y11 D=y22−y21 E=x21−x11 F=y21−y11

and get

k1 A − k2 B = E k1 C − k2 D = F

This can be written in matrix form:

[ A−B C−D ] [ k1 k2 ] = [ E F ]

Then we use Cramer's rule. With M as the coefficient matrix, we calculate the determinants

det(M) = |M| = A(−D) − C(−B) = BC−AD

If the value of this determinant is zero, then the equation system has no solution. Geometrically that means that the lines are parallel and distinct, they have no point in common.

det(M1) = |M1| = | E−B F−D | = E(−D) − F(−B) = BF−ED det(M2) = |M2| = | AE CF | = A<F−CE

Then

k1= |M1| |M| = BF−ED BC−AD k2= |M2| |M| = A<F−CE BC−AD

Resubstitution gives

k1 = (x22−x21) (y21−y11) − (x21−x11) (y22−y21) (x22−x21) (y12−y11) − (x12−x11) (y22−y21) k2 = (x12−x11) (y21−y11) − (y12−y11) (x21−x11) (x22−x21) (y12−y11) − (x12−x11) (y22−y21)

We can now check the value of k. If 0≤k≤1, then the intersection lies on the line segment. If k is outside of that range, the intersection point lies on the line outside of that segment.
